# University Management System

The University Management System (UMS) is a project that will manage the operations of a university. While this was a group effort, I took point on the **Student Management** role and was also responsible for debugging and mapping out our plan using pseudocode.

## Features
* Administrator: access administrator menu, add or remove course, remove student data, manage lecturers, generate summary report, and display all data.
* Lecturer:  access lecturer menu, view assigned module, record student grades, display student list, track student attendance, and view all student's grades.
* Student Management: access student management menu, view available courses, manage student enrollment, access attendance records, and access student grades.
* Registrar: access registrar menu, register new student, access and update student data, remove and assign student courses, and generate student transcript.
* Accountant: access accountant menu, manage student tuition fees, display student with outstanding fees, and view financial summary.
